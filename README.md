## Цель проекта
Интеграция новых источников данных и реализация дополнительных ETL/Streaming-процессов для расширения аналитических возможностей корпоративной платформы с использованием стека Yandex Cloud.

## Структура репозитория
* `/task1_data_transfer` — SQL/YQL скрипты для настройки YDB и работы с Data Transfer.
* `/task2_airflow` — DAG-файлы Apache AirFlow и PySpark-задания для обработки flat-файлов.
* `/task3_kafka` — Скрипты потоковой обработки данных из Apache Kafka (разворачивание JSON).
* `/task4_datalens` — Материалы и ссылки, касающиеся визуализации в DataLens.

## Отчет о проделанной работе

### Задание 1. Работа с Yandex DataTransfer
**Статус:** В процессе 

**Описание шагов:**
1. Развернута БД Managed Service for YDB.
2. Созданы исходные таблицы и подготовлен датасет `transactions_v2` (объем > 30 Мб).
3. Настроен эндпоинт источника (YDB) и эндпоинт приемника (Yandex Object Storage).
4. Запущен трансфер, данные успешно выгружены в бакет.



### Задание 2. Автоматизация Yandex Data Processing при помощи Apache AirFlow
**Статус:** В процессе 

**Описание шагов:**
1. Развернут кластер Managed Service for Apache Airflow.
2. Написан PySpark-скрипт для обработки плоских таблиц (parquet/csv).
3. Создан DAG, который автоматизирует жизненный цикл кластера Data Processing (создание -> запуск PySpark -> удаление).
4. Проведена обработка файла объемом > 50 Мб.



### Задание 3. Работа с топиками Apache Kafka и PySpark
**Статус:** В процессе 

**Описание шагов:**
1. Развернут кластер Managed Service for Apache Kafka, создан топик для стриминга.
2. Написано PySpark-задание (Structured Streaming) для чтения сообщений из топика.
3. Реализована логика парсинга: вложенный JSON (с полями `customer`, `loan`, `scoring`) раскладывается в плоскую структуру DataFrame.
4. Результат сохраняется в целевое хранилище.




### Задание 4. Визуализация в DataLens
**Статус:** В процессе 

**Описание шагов:**
1. Создано подключение к целевой базе данных в Yandex DataLens.
2. Подготовлены датасеты на основе таблиц из предыдущих шагов.
3. Построены дашборды для визуализации аналитики.
